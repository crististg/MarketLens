# Gemini Project Context: MarketLens

## Project Overview

"MarketLens" is an AI-powered market intelligence system designed to provide explainable insights into market movements by synthesizing market data, macroeconomic indicators, financial news, and cross-asset correlations.

*   **Frontend**: A Next.js (React) application built with TypeScript and Tailwind CSS.
*   **Backend**: Planned to be a Python 3.10+ FastAPI application with PostgreSQL, Redis, and SQLAlchemy. (Not yet implemented)
*   **Key Features**:
    *   AI-powered market analysis and explanations.
    *   Context-aware AI assistant.
    *   Integration with Alpha Vantage, FRED, CoinGecko, and Marketaux for data.

## Building and Running

### 1. Installation

Install dependencies:
```bash
npm install
```

### 2. Running for Local Development

To run the development server:
```bash
npm run dev
```
The application will typically be available at `http://localhost:3000`.

## Development Conventions

*   **Frontend Technologies**: Next.js (React), TypeScript, Tailwind CSS, ESLint.
*   **Code Style**: Adhere to ESLint rules configured in the project.
